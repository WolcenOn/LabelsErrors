Comparador didáctico XML ↔ XSD

Valida de forma simplificada si un XML encaja con un XSD básico: raíz, secuencia de elementos directos y tipos simples comunes. Sirve para practicar, no sustituye a un validador XSD completo.

Chuleta JSON

Objeto: { "clave": "valor" }

Array: [1, 2, 3]

Tipos: string, number, boolean, null, array, object.

No válido: claves sin comillas, comillas simples, comas finales, undefined, comentarios, saltos de línea literales dentro de strings.

Chuleta XML / XSD / XSLT

XML: una raíz única, etiquetas bien anidadas, atributos con comillas.

Escapado: &amp; para &, &lt; para < en texto.

XSD: declara xmlns:xs, usa tipos como xs:string, xs:integer, xs:decimal, xs:boolean.

XSLT: declara xmlns:xsl, cuida cierres como <xsl:value-of .../> y expresiones XPath.
